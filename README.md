# SLVoicePlayer
###SLVoicePlayer 是一个多功能的音乐播放器，持播放语音队列，播放单个音频，播放，暂停，上一首，下一首，快进，快退，缓存音频，下载音频等功能，目前v1.0.0版本，持续更新中
